## 1. 프로젝트 개요
> `MyCarForMe`는  처음부터 끝까지 원스톱으로 진행되는 중고차 거래 플랫폼으로,
 맞춤형 비교 분석과 실시간 배송 추적까지 제공합니다. 
 
 > 차량 검색부터 구매, 배송까지 최적화된 중고차 구매 경험을 실현할 수 있습니다.
